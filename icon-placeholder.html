Instructions

To create your PNG icon files:

  1. Take a screenshot of each icon above
  2. Crop them precisely to the icon boundaries
  3. Save them as PNG files in your /assets/icons/ directory with these filenames:
    • icon-192x192.png
    • icon-512x512.png

Alternatively, you can use an online SVG-to-PNG converter with this SVG code:
